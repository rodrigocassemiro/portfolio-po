# User Story - Visit Scheduling

## User Story

As a customer,

I want to schedule a property visit online,

So that I can choose a convenient date and time without contacting an agent manually.

## Acceptance Criteria

### Scenario 1

Given that I am interested in a property

When I select an available date and time

Then the visit should be successfully scheduled.

### Scenario 2

Given that a selected time slot is no longer available

When I attempt to schedule the visit

Then I should be prompted to choose another available slot.

### Scenario 3

Given that my visit is scheduled

When the appointment is confirmed

Then I should receive a confirmation notification.

## Business Value

- Reduced scheduling friction
- Improved customer convenience
- Increased visit bookings
- Better operational efficiency
- Higher conversion potential
