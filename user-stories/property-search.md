# User Story - Property Search

## User Story

As a customer,

I want to search and filter available properties,

So that I can quickly find options that match my preferences and rental needs.

## Acceptance Criteria

### Scenario 1

Given that I am searching for a property

When I apply location filters

Then I should only see properties available in the selected area.

### Scenario 2

Given that I am searching for a property

When I apply filters such as price, bedrooms or property type

Then the results should update according to my criteria.

### Scenario 3

Given that no properties match my filters

When the search is completed

Then I should receive a clear message and suggestions to adjust my filters.

## Business Value

- Faster property discovery
- Improved customer experience
- Increased engagement
- Higher lead generation
- Better conversion rates
